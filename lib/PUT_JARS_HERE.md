# Drop your JARs here

Download and place the following JAR files in this folder:

## JavaFX 21 (from https://gluonhq.com/products/javafx/)
Download the JavaFX 21 SDK for Windows, extract it, and copy these from the `lib/` folder inside the SDK:
- javafx-controls.jar
- javafx-fxml.jar
- javafx-graphics.jar
- javafx-base.jar

## MySQL Connector/J (from https://dev.mysql.com/downloads/connector/j/)
Download "Platform Independent" ZIP, extract and copy:
- mysql-connector-j-8.x.x.jar  →  rename to  mysql-connector-j.jar

Once all JARs are in this folder, open the project in IntelliJ and everything will resolve automatically.
